# Calculadora Científica 🔢

Proyecto desarrollado en Python como parte del curso de Ingeniería de Sistemas - UNTELS.

## ¿Cómo ejecutar?

```bash
python calculadora.py
```

## Funcionalidades

### Operaciones básicas
- Suma, Resta, Multiplicación, División
- Potencia
- Raíz cuadrada

### Operaciones científicas
- Seno, Coseno, Tangente (en grados)
- Logaritmo base 10
- Logaritmo natural (ln)
- Factorial
- Valor absoluto

## Versiones
- **v1.0** - Operaciones básicas (suma, resta, multiplicación, división)
- **v2.0** - Operaciones científicas completas (trigonometría, logaritmos, factorial)

## Tecnología
- Python 3.x
- Librería `math` (estándar)
